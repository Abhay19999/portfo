003 : Image converter

A program which helps someone to convert thousands of images at a same time within seconds.

"When there is a need of converting thousands of images from one format to another a python based image converter can be used to overcome such difficulties within few seconds."

Project defination
Enter the folder containing the images and a new folder in which the converted images needs to be saved.
After hitting enter, a new folder will be created with converted images in it and a message will be printed for each image.
